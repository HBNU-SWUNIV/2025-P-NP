## <u>Teamate</u> Project Background
- ### 필요성
  - 지뢰찾기 문제는 단순한 규칙을 지니고 있음에도 불구하고, 경우의 수가 기하급수적으로 증가하는 NP-완전 문제로 분류되어 효율적인 해결이 어렵다.
  - 기존 지뢰찾기 AI 모델들은 불확실성 상황에서의 합리적 추론이 제한적이며, 지뢰 여부를 이진적으로만 예측하여 확률적 의사결정 과정을 충분히 반영하지 못한다.
  - 본 연구는 양자역학의 중첩(Superposition) 개념을 모방하여, 미개방 칸을 확률 상태로 정의하고 관측(클릭) 에 따라 확률이 갱신되는 방식을 적용함으로써 불확실성 추론의 정량화를 가능하게 한다.
- ### 기존 해결책의 문제점
  - 완전 탐색 방식은 계산 비용이 지나치게 크며, 실제 실시간 해결에 적합하지 않다.
  - CNN 기반 패턴 예측 모델은 공간 패턴은 학습하나, 인접 칸 간의 조건부 확률 관계를 구조적으로 반영하는 데 한계가 있다.
  - 기존 연구는 불확실성을 단순 미확정 상태로 처리할 뿐, 확률 갱신 및 수렴 과정을 다루지 못한다.
  
## System Design
  - ### System Requirements
    - 지뢰찾기 10x10 보드를 기반으로 각 칸을 확률적 중첩 상태로 표현하는 확률 추론 모델이 필요하다.
    - CNN, GNN, 베이지안 네트워크 등 다양한 AI 모델과의 성능 비교 실험이 가능해야 한다.
    - 탐색 과정에서 확률 정보를 사용하여 강화학습 기반 클릭 전략을 수행할 수 있어야 한다.
    - Minesweeper GUI에서 확률 분포 시각화 및 관측 후 갱신 과정이 실시간으로 표현되어야 한다.
    
## Case Study
  - ### Description
  - 1.데이터수집
    - 10x10 보드에서 다양한 초기 클릭 전략을 기반으로 플레이 데이터를 수집하였다.
    - 각 클릭 순간의 인접 숫자, 실제 지뢰 여부, 추론 확률 등을 기록하였다.
  - 2. 확률 기반 중첩 상태 모델링
    - 모든 칸의 초기 상태를 지뢰/안전의 확률이 동등한 중첩 상태 (P=0.5) 로 정의하였다.
    - 사용자가 칸을 클릭(관측)하면, 실제 결과에 기반하여 주변 칸 확률을 조건부 확률로 갱신하였다.
  - 3. AI 모델 비교
    - CNN: 보드 패턴 학습 후 칸별 지뢰 확률 예측.
    - GNN: 보드를 그래프 구조로 정의하여 인접 칸 간 확률 메시지 전달.
    - 베이지안 네트워크: 인접 칸 숫자를 기반으로 조건부 확률 업데이트.
  - 4. 강화학습 결합
    - 확률 추론 모델이 제공하는 안전 확률이 높은 칸을 우선적으로 선택하는 전략을 강화학습을 통해 학습.
    - 게임 성공률을 최대화하도록 보상 함수 설계.
  
  
## Conclusion
  - ### 연구의의
  - 본 과제는 NP-완전 문제 해결 과정에서 불확실성을 확률적으로 처리하는 새로운 접근을 제시한다.
  - 기존 단순 분류 중심 연구와 달리, 중첩 → 관측 → 수렴 과정을 모델링하여 의사결정 합리성을 향상시킨다.
  - ### 활용 및 확장 가능성
  - 본 연구 결과는 단순한 게임 AI를 넘어 물류 경로 최적화, 회로 배선 문제, 자원 스케줄링 등 불확실성 기반 최적화 문제에 적용할 수 있다.
  - 확률 추론 + 강화학습 구조는 특허 및 실용신안 출원 가능성이 있으며, 시각화 가능한 데모는 학술 발표 및 캡스톤 전시 등에 활용할 수 있다.
